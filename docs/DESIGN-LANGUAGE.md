# IGLOUE Design Language

## Philosophy

Every screen should feel calm.

Customers arrive because they are hot, uncomfortable or under pressure.

The website should reduce stress, not add to it.

The design should feel effortless.

---

# Design Principles

Less but better.

Whitespace is a feature.

Every element must have a purpose.

Never decorate for the sake of decoration.

Never overwhelm.

---

# Visual Hierarchy

Each screen should answer ONE primary question.

Everything else supports that answer.

Example:

Hero

Question:
Can IGLOUE help me?

Answer:
Yes.

CTA:
Check your postcode.

---

# Buttons

Primary

Blue

Rounded

High contrast

Always one main CTA per screen.

Secondary

White

Bordered

Lower emphasis.

---

# Cards

Every card should have:

Rounded corners

Soft shadow

Large padding

Simple typography

Clear hierarchy

---

# Icons

Simple.

Outlined.

Consistent stroke width.

Blue accent colour.

Never cartoonish.

---

# Photography

Bright.

Natural.

Real homes.

Real products.

No fake smiling models.

No exaggerated marketing photos.

---

# Animations

Fast.

Smooth.

Purposeful.

No spinning.

No bouncing.

No flashy effects.

Animations should communicate quality.

---

# Layout

Every page should follow:

Hero

↓

Decision

↓

Information

↓

Trust

↓

Action

---

# Trust Signals

Transparent pricing.

Real delivery area.

Real photos.

Real reviews.

Simple language.

No hidden costs.

---

# Golden Rule

If removing something improves clarity,

remove it.
