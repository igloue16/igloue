# IGLOUE Mobile Workflow

## Purpose

Allow small updates to the IGLOUE project from a phone while keeping the PC repository clean and safe.

---

## Best Use From Phone

Phone edits are useful for:

- documentation
- prices
- French copy
- postcode lists
- small HTML text changes
- planning files

Phone edits are not ideal for:

- large JavaScript changes
- major layout changes
- CSS architecture
- debugging
- file reorganising

---

## Phone Workflow

1. Open GitHub in mobile browser
2. Edit the file
3. Commit directly to main
4. Check GitHub Pages after a short delay

---

## PC Workflow After Phone Edits

Before editing on PC:

1. Open GitHub Desktop
2. Click Fetch origin
3. Click Pull origin if available
4. Open VS Code
5. Continue work

---

## Golden Rule

Before starting work on any device, pull the latest version first.
