# IGLOUE Technical Plan

## Current Phase

Phase 1 is a static GitHub Pages website.

## Rules

- Mobile-first
- Semantic HTML
- CSS variables
- Modular JavaScript
- Accessibility first
- SEO friendly
- No customer passwords in GitHub
- No card details in GitHub
- No private admin tools in GitHub Pages

## Future Architecture

- Public website: GitHub Pages
- Payments: Stripe Checkout or another secure payment provider
- Bookings: secure backend or trusted booking provider
- Customer portal: future authenticated application
- Admin dashboard: protected backend, never public static files
